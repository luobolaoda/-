### 4.3 热力学自洽性

**理论模型**：
```
阳指令（存入）：E_阳 = E_动态 + E_静态
阴指令（释放）：E_阴 = η × E_节点电容

约束：
1. E_阴 < E_阳（回收能量小于消耗能量）
2. η < 1（回收效率小于100%）
3. 净熵增 ΔS = (E_阳 - E_阴) / T > 0
```

## 五、核心公式总结

### 5.1 太极三态基本定义

| 状态 | 符号 | 数值 | 电压 | 物理含义 |
|------|------|------|------|---------|
| 阴 | Y | -1 | 0V | 负极性、释放能量 |
| 太极 | T | 0 | 0.9V | 中性态 |
| 阳 | X | +1 | 1.8V | 正极性、存储能量 |

### 5.2 核心运算公式

**单trit加法（同构于Z/3Z）**：
```
阴 + 阴 = 阴
阴 + 太极 = 阴
阴 + 阳 = 太极      ← 核心公式
太极 + 任何 = 任何
阳 + 阳 = 阳
阳 + 阴 = 太极
```

**逆执机制**：
```
¬阴 = 阳
¬太极 = 太极
¬阳 = 阴
¬(a ⊕ b) = ¬a ⊕ ¬b    ← 逆执与加法交换
```

**能量守恒**：
```
a.value + b.value = sum.value + 3 × carry.value
```
