## 这是什么

太极三进制是一套从零设计的纯中文三进制计算体系。它不是对现有二进制计算机的改良，而是基于“阴/太极/阳”三态逻辑，重新定义进制、指令集、微架构和工具链的完整前沿研究。

核心创新：指令自带能量方向——阳指令消耗能量，阴指令回收能量，太极指令维持平衡。

---

## 设计哲学

二进制计算机只分0和1。太极三进制计算机分三种状态：

- 阴（-1）：收敛、回收、逆向
- 太极（0）：平衡、保持、不变
- 阳（+1）：释放、创造、正向

计算不只是数据的变换，也是能量的流动。

## 关于名称

“太极”取自中国哲学中阴阳平衡的概念。在这套体系里，太极不是“空”，而是阴阳之间的平衡态——既有阴的收敛倾向，又有阳的释放倾向，两者相互抵消，归于中和。
