2. **平衡三进制算术正确**
   - 1+(-1)=0 ✓
   - 全阳+全阴=0 ✓
   - 全阳+全阳=溢出（=-1）✓

   - 左移1位 = ×3（不是×2）
   - Word(1)<<3 = 27（1×3³）
   - 循环移位27位=自身

4. **寄存器设计正确**
   - R25(ZR) = 全太极（零寄存器）
   - R26(OR) = 全阳（壹寄存器）

5. **三段式内存设计正确**
   - 北极段：只读，状态不更新
   - 赤道段：可读写，状态标记为阳
   - 南极段：只写，状态不更新
